Why PWA?

and why you should care

Progressive Web Applications allow developers to create a single web application that is going to work in the browser, desktop, tablets, and mobile phones. This cuts down both development costs and time it takes to launch an application.

What can PWA do?

Progressive Web Applications are more capable than most people expect. They can already be installed as native applications, send push notifications, do a background data sync, and receive access to device functionalities, such as camera. Of course this varies depending on the browser & device, but in general PWA adoption is growing.

When to choose PWA?

There aren't any downsides in adopting PWA over more traditional web development. The real comparison is between native mobile applications and PWAs. The main limitations are what the device developers have set in place. An example of this would be that iOS currently has not implemented a functionality for PWAs to send push notifications.

Who even uses PWA?

A lot of people, actually, and you might not even know you're using them. Play Store (Android version of App Store) has been accepting PWAs without any modifications since 2019. Many companies have seen growth in SEO performance & conversion rates after adding PWA support. This has made the E-Commerce industry quickly adopt this technology. Some well known companies who are using PWAs include Instagram, TikTok, and many more!
